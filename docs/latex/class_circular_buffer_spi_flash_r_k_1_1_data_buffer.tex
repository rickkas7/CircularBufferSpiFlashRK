\doxysubsection{Detailed Description}
Class to hold a copy of data, either by pointer and length or a c-\/string. 

Uses a buffer allocated by new internally. When storing c-\/strings internally, the null is included in the buffer for efficiently using \doxylink{class_circular_buffer_spi_flash_r_k_1_1_data_buffer_a18493df360febe932a3d5c24d0e85861}{c\+\_\+str()} and operator const char \texorpdfstring{$\ast$}{*}().
