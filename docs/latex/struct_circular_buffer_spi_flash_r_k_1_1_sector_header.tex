\doxysubsection{Detailed Description}
Structure store at the beginning of each sector. 

This is separate from \doxylink{struct_circular_buffer_spi_flash_r_k_1_1_sector_common}{Sector\+Common} since we store information about each sector in the circular buffer. The magic bytes are necessary in flash but not in RAM, so not storing it in RAM saves 4 bytes per sector.
